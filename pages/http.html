Internet Term 7

HTTP

This page explains HTTP using its definition, role, and analogy.

Definition

HTTP (Hypertext Transfer Protocol) is a communication protocol used for transferring web data between clients and servers.

Role

It defines how requests are made and how responses are delivered on the web.

Analogy

Like placing an order and receiving your meal.
